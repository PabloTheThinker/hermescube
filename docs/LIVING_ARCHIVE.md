# Living Archive — chambers inside HermesCube

HermesCube is one durable warehouse file, but **many cooperating chambers**:

| Chamber | Role |
|---------|------|
| identity | Peer card, traits, relationships |
| doctrine | Beliefs + wisdom crystals |
| intent | Open focuses (prospective) |
| procedure | Forge/trajectory drafts + promote gate |
| associate | Engram hubs + colony + **dot-links** |
| narrative | Journey + session digests |
| catalog | Type/topic/entity index |

`manage action=pulse` runs all chambers once (no LLM). Any Hermes agent with
`memory.provider: hermescube` inherits a living, self-improving archive.
